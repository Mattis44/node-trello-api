
## NODE-TRELLO-API
A fully asynchronous node module for trello api in CommonJS.

## Usage
### Installation using NPM.
`npm i node-trello-api`
### Import and use
  ```js
  const trelloClient = require('node-trello-api')
 const Trello = new trelloClient("API_KEY", "API_TOKEN")
```
